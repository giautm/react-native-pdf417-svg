# react-native-pdf417-svg

react native component for pdf417 barcode

## Installation

```sh
npm install react-native-pdf417-svg
```

## Usage

```js
import { multiply } from "react-native-pdf417-svg";

// ...

const result = await multiply(3, 7);
```
